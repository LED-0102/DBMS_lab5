This is a backend made in rust. Please install cargo using rustup on your device. Then simply write cargo run on the terminal
Project will display register page
Also please make sure you have postgresql setup on your device. Please change the .env file constants according to you like your database username and password. Once made sure postgresql database is up and running, please load it using db.sql file. Then you may use the register page. After entering the values, check if those are entered into the database, if not check the console of your browser tab for 401 Conflict error, that means user already exists.
